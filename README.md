# dataMining_homework
数据分析与预处理课后作业
数据集使用的是visit-patterns-by-census-block-group.zip和wine-review.zip
具体做了数据数值型和描述型变量的数据分析与统计，使用到了条形图、直方图、盒图
数据的缺失值处理使用了四种方式完成

2020-05-17更新：
对Wine数据集导出频繁模式并进行关联规则挖掘，具体做的是：
